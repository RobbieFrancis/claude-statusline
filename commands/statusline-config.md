---
description: Configure statusline display options with interactive TUI
---

Launch the statusline configurator to customize your statusline display.

Run this command:
```bash
python3 ~/.claude/statusline-configurator.py
```

**Features:**
- Toggle display options on/off with switches
- Edit the statusline title
- See a live preview as you make changes
- Save or cancel with buttons or keyboard shortcuts (s/Esc)

**Note:** Requires the `textual` library. Install with: `pip3 install textual`
